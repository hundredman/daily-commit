# 📌 Algorithms
이 폴더는 알고리즘 문제 풀이를 정리하는 공간입니다. 다양한 온라인 저지(Online Judge) 플랫폼에서 문제를 풀고, 풀이 코드와 설명을 함께 기록합니다.

## 📂 폴더 구조
```bash
📂 algorithms
 ├── 📂 baekjoon       # 백준 문제 풀이
 ├── 📂 programmers    # 프로그래머스 문제 풀이
 ├── 📂 leetcode       # 리트코드 문제 풀이
 ├── 📂 swea           # 삼성 SW Expert Academy
 ├── 📂 codetree       # 코드트리 문제 풀이
 ├── 📜 README.md      # 알고리즘 폴더 개요
 └── 📜 solved.log     # 푼 문제 기록
```

## ✅ 문제 풀이 기록 방식
- 각 플랫폼별 폴더에 문제 풀이 파일을 저장합니다.
- 파일명은 `{문제번호}_{문제이름}.확장자` 형식을 따릅니다.
    - 예시: `1000_A+B.py`, `42839_소수찾기.java`
- 풀이 코드에는 핵심 아이디어 및 주석을 포함합니다.

## 📜 solved.log 활용법
- `solved.log` 파일에는 문제를 푼 날짜와 문제 정보를 기록합니다.
- 예시:
  ```
  2025-02-11 | 백준 | 1000 | A+B | 브론즈5
  2025-02-12 | 프로그래머스 | 42839 | 소수 찾기 | LV2
  ```

## 🚀 목표
- 꾸준히 문제를 풀면서 알고리즘 실력을 키웁니다.
- 다양한 유형의 문제를 해결하며 문제 해결 능력을 향상시킵니다.
- 코드 리뷰 및 기록을 통해 더 나은 코드를 작성합니다.
