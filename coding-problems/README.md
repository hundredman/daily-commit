# 📌 Coding Problems
이 폴더는 다양한 코딩 문제 풀이를 정리하는 공간입니다. 알고리즘 문제와 SQL 문제를 통합하여 사이트별로 관리하며, 문제 해결 능력을 키우고 실력을 향상시키는 것이 목표입니다.

## 📂 폴더 구조
```bash
📂 coding-problems
 ├── 📂 baekjoon       # 백준 문제 풀이
 ├── 📂 programmers    # 프로그래머스 문제 풀이
 ├── 📂 leetcode       # 리트코드 문제 풀이
 ├── 📂 swea           # 삼성 SW Expert Academy
 ├── 📂 codetree       # 코드트리 문제 풀이
 ├── 📂 kaggle         # 캐글 데이터 분석 및 SQL 문제 풀이
 ├── 📜 solved.log     # 전체 문제 풀이 기록
 └── 📜 README.md      # 폴더 개요
```

## ✅ 문제 풀이 기록 방식
- 사이트별 폴더에 문제 풀이 파일을 저장합니다.
- 파일명은 `{문제번호}_{문제이름}.확장자` 형식을 따릅니다.
  - 예시: `1000_A+B.py`, `42839_소수찾기.java`, `001_select_all_users.sql`
- 풀이 코드에는 핵심 아이디어 및 주석을 포함하여 학습 효과를 높입니다.

## 📜 solved.log 활용법
- `solved.log` 파일에는 문제를 푼 날짜와 문제 정보를 기록합니다.
- 예시:
  ```
  2025-02-11 | Baekjoon | 1000 | A+B | 브론즈5
  2025-02-12 | Programmers | 42839 | 소수 찾기 | LV2
  2025-02-13 | LeetCode | 1 | Two Sum | Easy
  2025-02-14 | Kaggle | SQL | 001 | 모든 사용자 조회
  ```

## 🚀 목표
- 꾸준한 문제 풀이를 통해 알고리즘 및 SQL 실력을 키웁니다.
- 다양한 유형의 문제를 해결하며 문제 해결 능력을 향상시킵니다.
- 코드 리뷰 및 기록을 통해 더 나은 코드를 작성합니다.
