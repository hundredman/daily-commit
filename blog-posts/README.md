# 📌 Blog Posts
이 폴더는 개발 관련 블로그 포스트를 저장하는 공간입니다. SQL, Spring, CS 등의 주제를 정리하고, 학습한 내용을 공유합니다.

## 📂 폴더 구조
```bash
📂 blog-posts
 ├── 📂 SQL            # SQL 관련 포스트
 ├── 📂 Spring         # Spring 관련 포스트
 ├── 📂 CS             # 컴퓨터 과학 개념 정리
 └── 📜 README.md      # 블로그 폴더 개요
```

## ✅ 작성 방식
- 마크다운(`.md`) 형식으로 작성합니다.
- 파일명은 `{YYYY-MM-DD}_제목.md` 형식을 따릅니다.
    - 예시: `2025-02-11_sql-index-optimization.md`
- 글의 구조는 다음과 같이 구성합니다.
  ```md
  # 제목
  
  ## 개요
  간략한 설명 작성
  
  ## 본문
  학습한 내용 정리
  
  ## 참고자료
  ```

## 🚀 목표
- 학습한 내용을 정리하여 지식을 체계적으로 관리합니다.
- 개발 경험과 노하우를 공유하며 성장합니다.
- 꾸준한 글쓰기를 통해 기술 문서 작성 능력을 향상시킵니다.
